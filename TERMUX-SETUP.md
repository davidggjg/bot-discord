# הרצת הבוט על הטלפון (Termux) - מדריך מלא

## לפני שמתחילים - חשוב מאוד

הרצה על טלפון אומרת שהטלפון צריך:
1. **להיות מחובר למטען כל הזמן** (אחרת הסולר ימות מהר)
2. **Wi-Fi שלא נכבה במצב שינה**: הגדרות → Wi-Fi → מתקדם → "השאר Wi-Fi דלוק במצב שינה" → תמיד
3. **כיבוי אופטימיזציית סוללה ל-Termux**: הגדרות → אפליקציות → Termux → סוללה → "ללא הגבלות" (unrestricted)

בלי שלושת אלה, הטלפון "יישן" ויהרוג את הבוט בלי שתבין למה.

## שלב 1: התקנת Termux

**אל תתקין מחנות Play** - הגרסה שם מיושנת ותקועה. תתקין מ-F-Droid:
1. הורד והתקן F-Droid: https://f-droid.org
2. בתוך F-Droid, חפש "Termux" והתקן

## שלב 2: התקנת הכלים הנדרשים

פתח את Termux והרץ שורה אחר שורה:

```bash
pkg update -y && pkg upgrade -y
pkg install nodejs-lts git tmux wget unzip -y
termux-setup-storage
```

(אם תופיע בקשת הרשאה לאחסון, אשר אותה)

## שלב 3: העברת הקבצים לטלפון

יש לך כמה אפשרויות, הקלה ביותר:

1. שלח לעצמך את קובץ ה-ZIP (וואטסאפ לעצמך / גוגל דרייב / טלגרם)
2. בטלפון, הורד אותו לתיקיית ההורדות
3. ב-Termux:

```bash
cd ~
mkdir -p bot && cd bot
unzip /sdcard/Download/discord-bot-system.zip
cd discord-bot-system
```

(אם הקובץ נמצא במקום אחר, עדכן את הנתיב בהתאם - אפשר לבדוק עם `ls /sdcard/Download/`)

## שלב 4: התקנת התלויות

```bash
npm install
```

זה ייקח כמה דקות בטלפון (איטי יותר ממחשב). תן לזה לרוץ עד הסוף.

## שלב 5: הרצה עם הגנה מפני שינה (tmux)

זה השלב הקריטי - בלי tmux, ברגע שתסגור את Termux הבוט ימות.

```bash
termux-wake-lock
tmux new -s bot
```

עכשיו אתה בתוך "מסך" וירטואלי שנשאר חי גם אם תסגור את Termux. בתוכו תרץ:

```bash
cd ~/bot/discord-bot-system
npm start
```

תראה: `🚀 הדשבורד פעיל בכתובת: http://localhost:3000`

## שלב 6: יציאה בלי להרוג את הבוט

לחץ: `Ctrl+B` ואז `D` (לא Ctrl+D!) - זה "מתנתק" מה-tmux בלי לעצור אותו.
אפשר עכשיו לסגור את Termux, להחליף אפליקציות, וכו'. הבוט ימשיך לרוץ.

## איך לחזור לראות את הלוגים / לעצור את הבוט

פתח Termux מחדש והרץ:

```bash
tmux attach -t bot
```

זה יחזיר אותך למסך שבו הבוט רץ. כדי לעצור אותו: `Ctrl+C`.

## איך לגשת לדשבורד

הדשבורד רץ בתוך הטלפון עצמו ב-`localhost:3000`. כדי לגשת לו:

**אופציה א' (הכי פשוט): מהדפדפן באותו טלפון**
פתח את כרום/Firefox בטלפון ולך ל: `http://localhost:3000`

**אופציה ב': ממחשב באותה רשת Wi-Fi**
1. ב-Termux הרץ: `ifconfig` או `ip addr` ומצא את כתובת ה-IP של הטלפון ברשת המקומית (משהו כמו `192.168.1.XX`)
2. במחשב, גש ל: `http://192.168.1.XX:3000`

## בעיה ידועה: termux-wake-lock לא מספיק

לפי הדיווחים העדכניים, גם עם wake-lock יש מקרים (בעיקר Samsung/Xiaomi) שהמערכת עדיין "מנקה" תהליכים ברקע. אם הבוט ממשיך למות:

1. הגדרות → סוללה → "Deep Sleep" / "Adaptive Battery" → כבה ל-Termux במפורש
2. ודא שב-Termux יש התראה קבועה (אם אין התראה, Android עלול לעצור את התהליך - זו לא תקלה, זה מנגנון הגנה של אנדרואיד)

## חשוב: זה פתרון זמני

זכור שזה אמור להיות פתרון גישור עד שיהיה לך השרת בחול ל-FiveM. כשהוא יהיה מוכן, פשוט תעביר את כל תיקיית `discord-bot-system` לשרת ותרוץ עליו `npm install && npm start` בדיוק באותו אופן - שום דבר בקוד לא צריך להשתנות.
